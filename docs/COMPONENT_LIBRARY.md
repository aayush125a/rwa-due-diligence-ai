## Components

### 1. Sidebar Navigation

Purpose:
Primary navigation for all business user actions.

Items:
- Dashboard
- My Assets
- New Submission
- Notifications
- Settings

---

### 2. Top Navigation Bar

Purpose:
Display page title and user controls.

Elements:
- Page Title
- Search
- Notifications
- User Profile

---

### 3. KPI Cards

Purpose:
Provide a quick overview of asset activity.

Cards:
- Total Assets
- Under Review
- Approved
- Rejected

---

### 4. Quick Action Card

Purpose:
Allow users to immediately create a new asset submission.

Primary CTA:
Create New Submission

---

### 5. Recent Submissions Table

Purpose:
Display recently submitted assets.

Columns:
- Asset Name
- Asset Type
- Submitted Date
- Status
- Risk Score
- Action

---

### 6. Status Badge

Types:
- Draft
- Under Review
- Approved
- Rejected

---

### 7. Risk Score Badge

Levels:
- Low
- Medium
- High

---

### 8. Empty State

Purpose:
Displayed when no submissions exist.

CTA:
Create First Submission
